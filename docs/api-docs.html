🔬 Molecular Docking 分子对接

POST /api/v1/docking/run

执行分子对接计算。支持批量对接、柔性对接和共价对接模式。返回结合能、结合位点和3D坐标。

ParameterTypeDescription
protein_pdb requiredstringPDB ID或上传的蛋白质结构文件
ligands requiredstring[]SMILES数组，最多1000个分子
modestringdocking模式: standard | flexible | covalent
exhaustivenessint搜索彻底性，默认32，范围8-256
curl -X POST https://api.drugmind.ai/api/v1/docking/run \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "protein_pdb": "1M17", "ligands": ["CC(=O)OC1=CC=CC=C1C(=O)O", "CC1=CC=C(C=C1)S(=O)(=O)N"], "mode": "standard", "exhaustiveness": 32 }'
GET /api/v1/docking/{job_id}/results

获取对接任务结果。包含每个配体的结合能(kcal/mol)、RMSD、3D坐标和可视化数据。

📊 ADMET Prediction

POST /api/v1/admet/predict

ADMET五维预测。覆盖吸收(Caco-2, HIA)、分布(BBB, PPB)、代谢(CYP450)、排泄(CL)、毒性(hERG, Ames, LD50)等25+终点。

ParameterTypeDescription
smiles requiredstring[]SMILES数组
endpointsstring[]指定预测终点，默认全部
confidencefloat置信度阈值，默认0.7
curl -X POST https://api.drugmind.ai/api/v1/admet/predict \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{"smiles": ["CC(=O)OC1=CC=CC=C1C(=O)O"]}' # Response { "absorption": { "caco2": -4.2, "hia": 0.92 }, "distribution": { "bbb": 0.15, "ppb": 85.3 }, "metabolism": { "cyp3a4_inhibit": 0.23 }, "toxicity": { "herg": 0.08, "ames": 0.12 } }

🎯 Virtual Screening 虚拟筛选

POST /api/v1/screening/run

从百万级化合物库中虚拟筛选。支持DrugCLIP深度学习模型和传统分子对接两种策略。返回Top-N候选化合物。

ParameterTypeDescription
target requiredstring靶点PDB ID或蛋白质序列
librarystring化合物库: chembl | zinc | enamine | custom
strategystring筛选策略: clip | docking | hybrid
top_nint返回Top N候选物，默认100

🕸️ Knowledge Graph 知识图谱

GET /api/v1/kg/search

查询药物知识图谱。支持实体搜索、关系查询、路径发现。整合DrugBank、ChEMBL、PubChem等20+数据源。

ParameterTypeDescription
query requiredstring搜索关键词或实体名称
typestring实体类型: drug | gene | disease | pathway
depthint关系深度，默认2，最大4
POST /api/v1/kg/drug-repurpose

药物重定位推荐。基于知识图谱推理，为给定疾病推荐已上市药物的新适应症。

🤖 Digital Twin 数字孪生

POST /api/v1/twin/create

创建药物项目数字孪生。模拟药物在虚拟患者体内的ADME过程，预测PK/PD曲线和临床试验成功率。

POST /api/v1/twin/{twin_id}/simulate

运行数字孪生模拟。支持剂量探索、给药方案优化、虚拟临床试验等场景。
