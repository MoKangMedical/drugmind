API Documentation

RESTful API，支持JSON格式，覆盖药物发现全流程

Base URL: https://api.drugmind.ai/v1
认证方式：所有API请求需在Header中携带 Authorization: Bearer YOUR_API_KEY。API Key可在控制台获取。
POST /docking/submit 提交分子对接任务

请求参数

参数类型说明
receptorstring受体PDB文件URL或PDB ID
ligandsstring[]配体SMILES数组
enginestring对接引擎: vina / diffdock / gnina
centerfloat[3]对接盒子中心坐标 [x,y,z]
sizefloat[3]对接盒子尺寸 [x,y,z]

示例请求

{
  "receptor": "1ABC",
  "ligands": ["CC(=O)OC1=CC=CC=C1C(=O)O", "CC1=CC=C(C=C1)S(=O)(=O)N"],
  "engine": "vina",
  "center": [12.5, 8.3, -4.2],
  "size": [20, 20, 20]
}

响应

{
  "task_id": "dock_20260428_001",
  "status": "submitted",
  "estimated_time": "15min"
}
POST /admet/predict ADMET性质预测

请求参数

参数类型说明
smilesstring[]化合物SMILES数组
endpointsstring[]预测端点 (可选，默认全部28个)

响应

{
  "results": [{
    "smiles": "CC(=O)OC1=CC=CC=C1C(=O)O",
    "predictions": {
      "herg_inhibition": {"value": 0.12, "risk": "low"},
      "hepatotoxicity": {"value": 0.34, "risk": "medium"},
      "solubility": {"value": -2.1, "unit": "logS"},
      "bioavailability": {"value": 0.78}
    }
  }]
}
POST /screening/start 启动虚拟筛选

请求参数

参数类型说明
targetstring靶点PDB ID或UniProt ID
librarystring化合物库: zinc / chembl / custom
top_kint返回Top-K结果 (默认100)
filtersobjectLipinski/Veber过滤条件
GET /kg/query 知识图谱查询

请求参数

参数类型说明
entitystring实体名称或ID
typestring实体类型: disease/drug/target/gene
depthint关系遍历深度 (1-3)
GET /tasks/{task_id} 查询任务状态

响应

{
  "task_id": "dock_20260428_001",
  "status": "completed",
  "results_url": "https://api.drugmind.ai/v1/results/dock_20260428_001"
}
