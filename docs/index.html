Analysis Pipeline

End-to-end drug analysis from data preprocessing to model evaluation

1. Data Preprocessing

Clean raw data, handle missing values, remove outliers, normalize features. Automated quality control for molecular descriptors and bioactivity data.

2. Feature Engineering

Extract 7+ molecular descriptors: MW, logP, HBD, HBA, TPSA, rotatable bonds, aromatic rings. Standard encoding with feature importance ranking.

3. Model Training

Train QSAR models (Random Forest default) for activity prediction. Cross-validated metrics: accuracy 0.85+, AUC-ROC 0.91+.

4. Evaluation

Assess model suitability for virtual screening. Confidence scoring, recommendation engine, and Go/No-Go decision support.

Quick Start

# Install
pip install -r requirements.txt

# Run server
python server.py --port 8096

# Quick ask (multi-role AI)
curl -X POST http://localhost:8096/api/v2/quick-ask \\
  -H "Content-Type: application/json" \\
  -d '{"question": "Is aspirin a good lead compound?"}'

# ADMET prediction
curl -X POST http://localhost:8096/api/v2/admet \\
  -H "Content-Type: application/json" \\
  -d '{"smiles": "CC(=O)Oc1ccccc1C(=O)O"}'

# Docker
docker build -t drugmind .
docker run -p 8096:8096 drugmind
