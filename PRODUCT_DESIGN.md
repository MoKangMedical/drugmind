# DrugMind 产品平台设计文档

## 产品定位
AI驱动的药物研发协作平台 — "药物研发的GitHub + Reddit"
让所有想制药的人，有一个专业的地方讨论、协作、推进项目。

## 核心能力矩阵

### 🔬 AI讨论
- 5个专业数字分身（药物化学/生物/药理/数据/项目）
- 实时团队讨论+共识形成
- Second Me用户分身接入
- 讨论记录公开/私密

### 🧪 分子工具
- ADMET快速评估（RDKit）
- 化合物管线管理
- 场景模板（靶点/先导/Go-No-Go）
- 分子可视化

### 👥 社群
- 公开讨论Feed流
- 标签/话题聚合
- 用户关注/收藏
- 微信登录

### 🔗 Second Me集成
- 用户数字分身创建
- 分身训练数据上传
- MCP协议连接
- 跨平台分身同步

## 页面架构

### Web端（主站）
1. **首页** — 产品介绍+最新讨论+活跃用户
2. **讨论广场** — Feed流+搜索+标签筛选
3. **AI团队** — 5个数字分身介绍+实时对话
4. **分子工具** — ADMET评估+场景模板+化合物管理
5. **我的空间** — 用户资料+我的讨论+分身管理
6. **Second Me** — 创建/训练/同步分身

### 微信小程序（轻量版）
1. **首页** — 精选讨论+快速提问
2. **讨论** — 列表+详情+回复
3. **AI问答** — 选择角色+提问+回复
4. **我的** — 登录+资料

### 技术架构
```
前端（SPA单页应用）
├── 网站：纯HTML/CSS/JS（零依赖，响应式）
├── 微信：WeChat Mini Program原生
└── APP：未来可打包为React Native

后端（FastAPI）
├── /api/v2/* — 业务API
├── /api/mcp — MCP协议（Second Me）
├── /ws/* — WebSocket实时
└── / — 静态前端

数据层
├── 用户系统（注册/登录/Profile）
├── 讨论引擎（Feed/搜索/标签）
├── AI引擎（分身/讨论/评估）
└── 化合物管理（管线/追踪）
```
